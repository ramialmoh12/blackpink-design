# BLACKPINK Bloom Arena — Feminine 2026 Fan Website

A soft, elegant, and interactive BLACKPINK fan website concept designed and developed by **Rami Almohamad**.

This version is different from the darker neon BLACKPINK design. It uses a brighter feminine style with pearl backgrounds, blush pink, lilac, champagne accents, soft shadows, glass cards, and member-focused animations.

## Live Project

After uploading the files to GitHub Pages, your site will be available at:

```text
https://ramialmoh12.github.io/blackpink-area/
```

## Main Features

- Feminine pearl/blush visual direction
- Responsive layout for desktop, tablet, and mobile
- Sticky glass navigation
- Mobile burger menu
- Smooth scrolling
- Reveal animations on scroll
- Animated falling petals in the hero section
- Interactive BLACKPINK member cards
- Member spotlight section that changes aura by member
- Gallery filter buttons
- Lisa x FIFA Sound “Goals” feature section
- Built-in official Spotify embed so visitors can play “Goals” without YouTube
- Animated counters
- Scroll-to-top button
- Front-end fan club form interaction
- Footer credit for Rami Almohamad

## How to Upload to GitHub Pages

1. Open your `blackpink-area` repository on GitHub.
2. Delete or replace the old files.
3. Upload the new files:
   - `index.html`
   - `style.css`
   - `app.js`
   - `README.md`
4. Click **Commit changes**.
5. Wait 1–3 minutes.
6. Refresh your GitHub Pages link.
